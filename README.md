# [sprintRead](https://manthee1.github.io/sprintRead/)

A web based speed reading app

---

## How to use

Just head on over to [sprintRead](https://manthee1.github.io/sprintRead/)

---

### Keys  

* [ESC] Back to Text input screen
* [◀️,▶️] Word forward/backward
* [🔼,🔽] WPM higher/lower
* [SPACE] Pause
